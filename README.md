## Minimal [(Free version)](https://minimal-kit-react.vercel.app/)

## Getting started

- Recommended `node js 14.x` and `npm 6+`
- Install dependencies: `npm install` or `yarn install`
- Start the project: `npm run start` or `yarn start`
